# Appointment System Application

## Requirements
- Python 3.x (any version works)
- Install pip (1. Run `curl -O https://bootstrap.pypa.io/get-pip.py` 2. python get-pip.py)
- `virtualenv` (if not already installed, use `pip install virtualenv`)

## Setup

### Step 1: Clone the repository
```bash
git clone https://github.com/shubmi02/Appointment-System-Application
cd Appointment-System-Application
```
### Step 2: Create VirtualEnv and Activate
```bash
python -m venv venv
source venv/bin/activate # used to activate the virtualenv
```

### Step 3: Install packages
```bash
pip install -r requirements.txt
```

### Step 4: Run Application
```bash
flask run
```

### Step 5: View Application
In a browser open http://127.0.0.1:5000
